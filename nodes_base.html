What is an Open Node?

The Monero software architecture allows for an easy way to use Monero without downloading the entire blockchain.

This is called "using a remote node". Essentially, your wallet program connects to someone elses monerod (blockchain) program.

It is somewhat slower, and there is some information leak, as detailed here.

That being said, no one can steal your monero because you use a remote node.


To use with monero-wallet-cli, just use these commands

Windows

monero-wallet-cli.exe --daemon-host ADDRESS.OF.HOST

Unix-type (ubuntu, MacOS)

./monero-wallet-cli --daemon-host ADDRESS.OF.HOST
